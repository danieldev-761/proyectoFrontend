# Frontend Project

Personal portfolio and pet gallery created as an educational project to practice web design with HTML and CSS.

## Description

This project contains a responsive personal portfolio and a secondary pets page. It includes sections for introduction, projects, contact, and a gallery with pet cards.

## Features

- Main page with navigation and projects section
- Projects section organized with `display: grid`
- Secondary pets page with card gallery
- Use of Flexbox and Grid for responsive layouts
- Modern styles with CSS variables and hover effects

## Project Structure

- `index.html` — main portfolio page
- `mascotas.html` — pets gallery page
- `styles.css` — shared styles for both pages
- `style.min.css` — minified css file
- `assets/images/` — profile images, background, and pets

## How to Use

1. Clone or download the repository.
2. Open `index.html` in a web browser.
3. Navigate through the portfolio sections and visit `pets.html` to see the gallery.

## Recommendations

- Make sure you have the correct images in `assets/images/`.
- You can customize the texts, colors, and cards with more projects or pets.
- To see the responsive design, reduce the browser width or test on a mobile device.

## Technologies

- HTML
- CSS
- Javascript

## Notes

This project is ideal for learning the basics of web layout and improving the use of CSS Grid and Flexbox in contemporary layouts.
