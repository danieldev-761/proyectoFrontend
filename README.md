# proyectoFrontend

Portafolio personal y galería de mascotas creada como proyecto educativo para practicar diseño web con HTML y CSS.

## Descripción

Este proyecto contiene un portafolio personal responsive y una página secundaria de mascotas. Incluye secciones de presentación, proyectos, contacto y una galería con tarjetas de mascotas.

## Características

- Página principal con navegación y sección de proyectos
- Sección de proyectos organizada con `display: grid`
- Página secundaria de mascotas con galería de tarjetas
- Uso de Flexbox y Grid para layouts responsivos
- Estilos modernos con variables CSS y efectos hover

## Estructura del proyecto

- `index.html` — página principal del portafolio
- `mascotas.html` — página de galería de mascotas
- `styles.css` — estilos compartidos para ambas páginas
- `assets/images/` — imágenes de perfil, fondo y mascotas

## Cómo usar

1. Clona o descarga el repositorio.
2. Abre `index.html` en un navegador web.
3. Navega entre las secciones del portafolio y entra en `mascotas.html` para ver la galería.

## Recomendaciones

- Asegúrate de tener las imágenes correctas en `assets/images/`.
- Puedes personalizar los textos, colores y tarjetas con más proyectos o mascotas.
- Para ver el diseño responsive, reduce el ancho del navegador o prueba en un dispositivo móvil.

## Tecnologías

- HTML
- CSS

## Notas

Este proyecto es ideal para aprender las bases de maquetación web y mejorar el uso de CSS Grid y Flexbox en layouts contemporáneos.
